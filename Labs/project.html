Project - Presenting English Soccer Team Rankings using R Shiny and Leaflet

The project for GEOG 774 was reasonably open-ended, requiring 2 new "functionalities". I decided to explore what could be done using R Shiny, mainly because I found a dataset of English soccer results and wanted to incorporate it in my project. My goal was to produce an interactive map that allowed the user to explore the dataset, including team's home-grounds. Functionalities not covered in the GEOG 774 course included: use of R Shiny, ability to interactively alter the colouring and legend for a covariate shown in the map, interactive plot being presented via a "modal" window, and toggles to add and remove features.

The map was completed with a number of open-source resources. Development was conducted using R studio, and included the use of the following libraries: "shiny", "leaflet", "shinyBS" (for the modal functionality), "RcolorBrewer" (for the colour palettes), and "ggplot2" for the plots rendered in the modal window. I also experimented with the "plotly" library to include mouse hover interaction in the plots, but the result was uglier than the pure ggplot2 result, so I removed it. The data for the team ranking came from James Curley's engsoccerdata R package (downloadable from here ). A large proportion of the home ground locations and capacities were obtained from here , with the missing teams (about 20) added manually via websearch.

The development of the webpage relied heavily on two sources. The shiny interactivity was developed by closely following Matthew Leonawicz's excellent Shiny leaflet example ( here ). The team analysis and plotting relied heavily on GDA of england . In particular, the code to compute and plot the team ranking was used directly. This was then adapted to produce the the average ranking over time, and the subsequent plot. This analysis was used to rank the English soccer teams over time. For those interested, the top 10 were: 1)Liverpool, 2)Arsenal, 3)Everton, 4)Manchester United, 5)Aston Villa, 6)Tottenham Hotspur, 7)Chelsea, 8)Manchester City, 9)Newcastle United, and 10)Sunderland.

The resulting R app was published via the shinyio.apps server. This service serves the resulting web app at no cost, which I then have embedded in this page using iframes. This approach is reasonable stable, although I have noticed the odd issue on occasion. If embedded web app is behaving badly you can visit the one served by shinyapps.io here .
