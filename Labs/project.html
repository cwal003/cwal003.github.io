Project - Presenting English Soccer Team Rankings using R Shiny and Leaflet

The project for GEOG 774 was reasonably open-ended, requiring 2 new "functionalities". I decided to explore what could be done using R Shiny, mainly because I found a dataset of English soccer results and wanted to incorporate it in my project. My goal was to produce an interactive map that allowed the user to explore the dataset, including team's home-grounds. Functionalities not covered in the GEOG 774 course included: use of R Shiny, ability to interactively alter the colouring and legend for a covariate shown in the map, interactive plot being presented via a "modal" window, and toggles to add and remove features. The map was completed with a number of open-source resources. Development was conducted using R studio, and included the use of the following libraries: "shiny", "leaflet", "shinyBS" (for the modal functionality), "RcolorBrewer" (for the colour palettes), and "ggplot2" for the plots rendered in the modal window. I also experimented with the "plotly" library to include mouse hover interaction in the plots, but the result was uglier than the pure ggplot2 result, so I removed it.
